# Vihara Market Labs

## Vision

建立一个长期的个人研究与实验平台（Market Lab），系统研究市场机制（Market Design）、市场微观结构（Market Microstructure）与交易所设计（Exchange Design）。

长期目标：

* 深入理解现代市场的底层运行机制
* 通过实验验证市场设计理论
* 构建自己的 Market Design Research Portfolio
* 为未来创业、研究或 PhD（例如东京大学 Kojima 教授方向）积累成果

---

# Why This Project

过去的职业经历：

* Options Market Making
* Futu Japan Launch
* OKX Core Trading

积累了大量市场基础设施和交易系统经验。

相比于：

* Alpha挖掘
* 因子研究
* 回测优化

更感兴趣的是：

* 市场为什么这样设计
* 交易所为什么这样运作
* 不同市场机制会带来什么结果
* 是否存在更优的市场结构

因此未来重点不再是：

> 如何预测市场

而是：

> 如何设计市场

# Long-Term Roadmap

知识地图（Knowledge Map）：

Level 0 → Market Foundations

* Why Markets Exist
* Information Aggregation
* Market Efficiency

Level 1 → Mechanism Design

* Incentive Compatibility
* Strategy Proofness
* Efficiency
* Stability

Level 2 → Auction Design

* Call Auction
* Continuous Auction
* Frequent Batch Auction
* Order Flow Auction
* MEV Auction

Level 3 → Matching Theory

* Stable Matching
* Deferred Acceptance
* Matching with Constraints
* Matching with Contracts

Level 4 → Market Microstructure

* Order Book
* Liquidity
* Price Discovery
* Adverse Selection
* Inventory Risk

Level 5 → Exchange Design

* Exchange Architecture
* Market Structure
* Prediction Markets
* Stock Token Markets
* Intent-based Trading Systems

注意：

这些是知识地图（Map）。

不是学习顺序。

学习顺序应由项目驱动。

# Learning Philosophy

错误路线：

Reading
→ Reading

→ Reading

正确路线：

Project
→ Questions
→ Papers
→ Experiments
→ Findings
→ New Questions

核心原则：

每读一篇重要论文，必须产出一个实验。
